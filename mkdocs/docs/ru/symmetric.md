# –°–∏–º–º–µ—Ç—Ä–∏—á–Ω–∞—è –∫—Ä–∏–ø—Ç–æ–≥—Ä–∞—Ñ–∏—è

## –í–≤–µ–¥–µ–Ω–∏–µ

–í –Ω–∞—Å—Ç–æ—è—â–µ–µ –≤—Ä–µ–º—è –≤—Å–µ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–µ –∫—Ä–∏–ø—Ç–æ—Å–∏—Å—Ç–µ–º—ã –ø—Ä–∏–Ω—è—Ç–æ —Ä–∞–∑–¥–µ–ª—è—Ç—å –Ω–∞ –¥–≤–∞ –∫–ª–∞—Å—Å–∞: _—Å–∏–º–º–µ—Ç—Ä–∏—á–Ω—ã–µ_ –∏ _–∞—Å–∏–º–º–µ—Ç—Ä–∏—á–Ω—ã–µ_.

–°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ, –≥–æ–≤–æ—Ä—è—Ç –æ —Å–∏–º–º–µ—Ç—Ä–∏—á–Ω–æ–π –∫—Ä–∏–ø—Ç–æ–≥—Ä–∞—Ñ–∏–∏ –∏ –∞—Å–∏–º–º–µ—Ç—Ä–∏—á–Ω–æ–π –∫—Ä–∏–ø—Ç–æ–≥—Ä–∞—Ñ–∏–∏.

__–í —Å–∏–º–º–µ—Ç—Ä–∏—á–Ω—ã—Ö –∫—Ä–∏–ø—Ç–æ—Å–∏—Å—Ç–µ–º–∞—Ö__ –æ–¥–Ω–∏–º –∏ —Ç–µ–º –∂–µ —Å–µ–∫—Ä–µ—Ç–Ω—ã–º –∫–ª—é—á–æ–º –æ—Å—É—â–µ—Å—Ç–≤–ª—è–µ—Ç—Å—è –∏ —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏–µ, –∏ —Ä–∞—Å—à–∏—Ñ—Ä–æ–≤–∞–Ω–∏–µ:

$$E_k(P) = C,
D_k(C) = P$$

–≥–¥–µ $–ï$ ‚Äì —Ñ—É–Ω–∫—Ü–∏—è –∑–∞—à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è, $k$ ‚Äì –∫–ª—é—á, $P$ ‚Äì –æ—Ç–∫—Ä—ã—Ç—ã–π —Ç–µ–∫—Å—Ç, $–°$ ‚Äì —à–∏—Ñ—Ä—Ç–µ–∫—Å—Ç,
$D$ ‚Äì —Ñ—É–Ω–∫—Ü–∏—è —Ä–∞—Å—à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è.

–ü—Ä–∏ —ç—Ç–æ–º —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤–æ —Å–ª–µ–¥—É—é—â–µ–µ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ:
$D_k(E_k(P)) = P$

### –ü–æ—Ç–æ–∫–æ–≤—ã–µ –∏ –±–ª–æ—á–Ω—ã–µ —à–∏—Ñ—Ä—ã

–ê–ª–≥–æ—Ä–∏—Ç–º—ã, –∫–æ—Ç–æ—Ä—ã–µ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—é—Ç –æ—Ç–∫—Ä—ã—Ç—ã–π —Ç–µ–∫—Å—Ç –ø–æ–±–∏—Ç–æ–≤–æ (–ø–æ–±–∞–π—Ç–æ–≤–æ), –Ω–∞–∑—ã–≤–∞—é—Ç __–ø–æ—Ç–æ–∫–æ–≤—ã–º–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–∞–º–∏__ –∏–ª–∏ –ø–æ—Ç–æ–∫–æ–≤—ã–º–∏ —à–∏—Ñ—Ä–∞–º–∏.

–ê–ª–≥–æ—Ä–∏—Ç–º—ã, –∫–æ—Ç–æ—Ä—ã–µ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—é—Ç –≥—Ä—É–ø–ø—ã –±–∏—Ç–æ–≤ (–±–ª–æ–∫–∏) –æ—Ç–∫—Ä—ã—Ç–æ–≥–æ —Ç–µ–∫—Å—Ç–∞, –Ω–∞–∑—ã–≤–∞—é—Ç __–±–ª–æ—á–Ω—ã–º–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–∞–º–∏__ –∏–ª–∏ –±–ª–æ—á–Ω—ã–º–∏ —à–∏—Ñ—Ä–∞–º–∏.

–î–æ–ª–≥–æ–µ –≤—Ä–µ–º—è –≤ –∫–æ–º–ø—å—é—Ç–µ—Ä–Ω—ã—Ö –∞–ª–≥–æ—Ä–∏—Ç–º–∞—Ö —Ç–∏–ø–∏—á–Ω—ã–π —Ä–∞–∑–º–µ—Ä –±–ª–æ–∫–∞ –±—ã–ª —Ä–∞–≤–µ–Ω 64 –±–∏—Ç–∞–º. –≠—Ç–æ –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –±–æ–ª—å—à–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, —á—Ç–æ–±—ã –∑–∞—Ç—Ä—É–¥–Ω–∏—Ç—å –∞–Ω–∞–ª–∏–∑, –∏ –≤ —Ç–æ –∂–µ –≤—Ä–µ–º—è –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –º–∞–ª–æ–µ, —á—Ç–æ–±—ã –±—ã—Ç—å —É–¥–æ–±–Ω—ã–º –¥–ª—è —Ä–∞–±–æ—Ç—ã. –í –Ω–∞—Å—Ç–æ—è—â–µ–µ –≤—Ä–µ–º—è –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è 128-—Ä–∞–∑—Ä—è–¥–Ω—ã–µ –±–ª–æ–∫–∏, —Ç–∞–∫ –∫–∞–∫ –¥–ª–∏–Ω–∞ –±–ª–æ–∫–∞ –≤ 64 –±–∏—Ç–∞ –Ω–µ —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è–µ—Ç —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–º —Ç—Ä–µ–±–æ–≤–∞–Ω–∏—è–º —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏ –∏ –Ω–∞–¥–µ–∂–Ω–æ—Å—Ç–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤.

–ù–∞–∏–±–æ–ª–µ–µ —à–∏—Ä–æ–∫–æ –ø—Ä–∏–º–µ–Ω—è–µ–º—ã–º–∏ –Ω–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ —Å–∏–º–º–µ—Ç—Ä–∏—á–Ω—ã–º–∏ –∫—Ä–∏–ø—Ç–æ—Å–∏—Å—Ç–µ–º–∞–º–∏ –¥–æ–ª–≥–æ–µ –≤—Ä–µ–º—è —è–≤–ª—è–ª–∏—Å—å —Å–∏—Å—Ç–µ–º—ã DES (—Å—Ç–∞–Ω–¥–∞—Ä—Ç –°–®–ê), IDEA (–µ–≤—Ä–æ–ø–µ–π—Å–∫–∏–π —Å—Ç–∞–Ω–¥–∞—Ä—Ç), –ì–û–°–¢ (—Å—Ç–∞–Ω–¥–∞—Ä—Ç –†–§) –∏ –∏—Ö –º–æ–¥–∏—Ñ–∏–∫–∞—Ü–∏–∏.
